# Typical applications



## Summary

   [![stable](http://badges.github.io/stability-badges/dist/stable.svg)![version](https://img.shields.io/badge/version-1.5.11-green)](http://github.com/badges/stability-badges)

OrbbecSDK_ROS2 is a wrapper for the Orbbec 3D camera that provides seamless integration with the ROS 2 environment. It
supports ROS 2 Foxy, Humble, and Jazzy distributions.

&nbsp;

For the usage  of orbbecSDK, please refer to the link for more detailed introduction:

[https://github.com/orbbec/OrbbecSDK-dev](https://github.com/orbbec/OrbbecSDK-dev)

&nbsp;

## Applications

* Depth Sensing
* Object Detection
* Body Tracking
* Positional Tracking
* Geo Tracking
* Spatial Mapping
* Camera Control
* Plane Detection
* Multi Camera Fusion
